Network Lab: VLANs, Inter-VLAN Routing, and Firewall ACLs

Cisco Packet Tracer lab with two access switches, one router-on-a-stick, and an edge firewall.

Download Switch1 Config Download Switch2 Config Download Router Config Download Firewall ACLs

Topology

  • VLAN 10: Users (192.168.10.0/24)
  • VLAN 20: Servers (192.168.20.0/24)
  • VLAN 30: Management (192.168.30.0/24)
  • ROAS on Router G0/0 subinterfaces (.1 gateways)
  • Edge firewall between Router G0/1 and Internet (simulated)

Goals

  1. Create VLANs and assign access ports on both switches.
  2. Configure trunk between switches and to the router.
  3. Configure router-on-a-stick for inter-VLAN routing and DHCP pools.
  4. Apply firewall ACLs to restrict traffic between VLANs and outbound.

Verification

  • From a VLAN 10 host, ping its gateway (192.168.10.1) and a VLAN 20 server.
  • Ensure DHCP leases are obtained per VLAN.
  • Test that blocked ports (e.g., SMB) are denied by ACLs.
  • Allow HTTPS to Internet via firewall; HTTP optionally blocked.
